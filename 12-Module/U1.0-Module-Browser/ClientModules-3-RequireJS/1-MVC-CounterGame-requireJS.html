MVC Counter with OO-Model and requireJS

Count: 

    here
    Counter.prototype.countUp = function() {console.log(this)}
    -> Counter is not accessible -> no "global namespace pollution"
    BUT
    require(['./game/Counter'], function (Counter) {Counter.prototype.countUp = function() {console.log(this)}})
    Counter is accessible through the require function (keeps a map of Module exports)
