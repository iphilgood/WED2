  • gulp-sass: "gulp sass:watch" übersetzt alles im Ordner sass.

Sass Variablen

Der article-tag sollte wie folgt gestyled werden mit Sass - nutzen Sie Sass Variablen und Sass Build-In-Funktionen:

  • Schriftfarbe: RGB Werte: 255,60,80.
  • Hintergrund: Komplementär Farbe von der Schrift.
  • Schriftgrösse sollte 50% grösser sein, als die Standardschriftfarbe.

Das generierte CSS sollte wie folgt aussehen:

article {
  color: #ff3c50;
  background: #3cffeb;
  font: 1.5rem;
}

Sass Mixing 1

Sie sollen ein mixing erstellen welche die Spanische-Fragezeichen ergänzt: ¿adónde vas?

Das Mixing sollte wie folgt genutzt werden können:

.symbol{
   @include addSymbol(after, "?");
   @include addSymbol(before, "¿");
}

Das resultierten CSS könnte wie folgt aussehen:

@charset "UTF-8";
.symbol:after {
  content: "?";
}
.symbol:before {
  content: "¿";
}

Sass Mixing 2

Auf http://cssarrowplease.com/ kann man CSS-Arrows erstellen.

  • Analysieren Sie die Änderungen je nach Richtung.
  • Erstellen Sie ein Sass Mixing, welches das gleiche Erreicht.
  • Das Mixing soll ein Partial sein.
  • Erstellen Sie ein SCSS File welches das Partial verwendet um 4 CSS Klassen zu generieren: .arrow-left .arrow-right .arrow-top und .arrow-bottom
  • Vermeiden Sie copy & paste.
  • Testen Sie ihr Lösung in einem HTML.
  • Vergleichen Sie Ihre Lösung mit der Musterlösung.

Eine Mögliche Lösung: styles.scss und _arrow.scss

Würde es Sinn machen Ihre Lösung mit Sass Vererbung zu erstellen?

Nein, da Parameter benutzt werden - dies ist mit Vererbung nicht möglich.
