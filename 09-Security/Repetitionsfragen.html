XSS

  1. Beschreiben Sie in wenigen Worten, was Cross-Site-Scripting ist.

    Cross-Site-Scripting ist eine Injection Attacke. Ein Angreifer versucht JavaScript-Code in ein System einzuschleusen, sodass dieser bei andern Benutzern geladen und ausgeführt wird.

  2. Welche der folgenden Angriffe kann ein Angrifer mit XSS durchführen?
    • Zugriff auf das lokale Dateisystem
    • Zugriff auf Kamera und Mikrofon
    • Auslesen der Browser-History
    • Öffnen einer P2P-Verbindung
    • Speichern von Daten im Browser
    • Auslesen des Session-Cookie
    • Lokalisierung des Clients (Position auslesen)
    • Zugriff auf Kamera und Mikrofon (Je nach Browsereinstellung nur mit Erlaubnis)
    • Öffnen einer P2P-Verbindung (z.B. WEBRTC)
    • Speichern von Daten im Browser (Localstorage)
    • Auslesen des Session-Cookie (Nur wenn kein HttpOnly Cookie)
    • Lokalisierung des Clients (Position auslesen) (Je nach Browsereinstellung nur mit Erlaubnis)

Eval

  1. Welche Gefahren birgt Eval auf dem Server und auf dem Client?

    Eval erlaubt das Auführen von beliebigem Code. Analog XSS kann ein Angreifer auf dem Client sämmtliche Funktionen nutzen, die der Browser anbietet. Auf dem Server kann der Angreifer entsprechend Funktionen und Variablen benutzen, die Node zur Verfügung stellt.

  2. Welche der folgenden Angriffe kann ein Angrifer mit Eval auf dem Node-Server durchführen?
    • Zugriff auf das lokale Dateisystem
    • Auslesen der Browser-History
    • Öffnen einer P2P-Verbindung
    • Auslesen der Session
    • Zugriff auf das Dateisystem des Benutzers
    • Zugriff auf Server-Konfiguration
    • Zugriff auf angebundene Datenbanken
    • Zugriff auf das lokale Dateisystem
    • Auslesen der Session
    • Zugriff auf Server-Konfiguration
    • Zugriff auf angebundene Datenbanken
  3. Warum gibt es in JavaScript und PHP "eval()" und in Java nicht?

    Java ist eine kompilierte Sprache und erlaubt kein Erzeugen von Code zur Laufzeit. PHP und JavaScript als interpretierte Sprachen ermöglichen dies einerseits von der technischen Seite her und bieten es andererseits über die Eval-Funktion explizit an.
