⇦ WED2 Übung W10: Layering + JS OO

Ziele der Übung

In dieser Übung werden Sie einen bestehenden JavaScript Source Code überarbeiten und schrittweise objekt-orientierte Ansätze in den Code einbringen (=Refactoring). Sie werden dazu angehalten, Ihre Lösungen aus den jeweiligen Aufgaben nach und nach zu erweitern sowie über die getätigten Verbesserungen zu reflektieren. Falls Sie Anhaltspunkte fürs Lösen der Aufgaben benötigen, können Sie jederzeit die beiliegenden Lösungen durchschauen.
Bitte beachten Sie, dass neben der gegebenen Lösungen andere Lösungen möglich sind, welche auch gut oder sogar besser (vollständiger) sein können. Fürs bessere Verständnis des Stoffes ist es ratsam, die in der Vorlesung behandelten Konzepte zumindest einmalig in den untenstehenden Übungen anzuwenden.
